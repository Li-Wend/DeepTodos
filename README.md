# Deeptodo

## 开发前准备
#### 1.1 安装 python 官网：https://www.python.org/
#### 1.2 设置 python 环境变量 (安装时默认勾选自动设置环境变量)
#### 2.1 安装 Node.js
#### 3.1 安装 Git 官网：https://github.com/
#### 4.1 安装 VSCode 开发工具 官网：https://code.visualstudio.com/
#### 4.2 安装 VSCode extensions - python、git 等
#### 5.1 下载 git repository 到电脑指定文件目录
#### 6.1 安装依赖 (使用 requirements.txt)
##### - pip install -r requirements.txt

## 运行

### 启动应用：
#### VSCode 中运行 python 文件 app.py 

### 访问应用：
#### http://localhost:5000

### 提交代码：
#### - 创建分支 如： dev
#### - 代码更改完成后在根目录 ./deeptodo 中打开 Terminal
#### - git status // 查看 git 状态
#### - git checkout dev // 切换到分支 dev
#### - git add . 
#### - git commit -m "message"
#### - git push origin dev
#### - merge dev to main branch
